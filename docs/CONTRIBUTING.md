# Contributing

Thanks for your interest in pgfusion!

## Bugs & Feature Requests

Open an issue on GitHub — bug reports and feature requests are welcome and the best way to contribute right now.

When filing a bug, include:
- pgfusion version / git commit
- PostgreSQL version and OS
- Steps to reproduce
- Expected vs actual output

## Pull Requests

PRs are not yet open. We're stabilising the core before accepting external contributions. Watch the repo — we'll announce when PRs are welcome.
